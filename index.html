📱 安装为原生应用
iOS · macOS · Android · Windows
⚠ 重要：浏览器只允许从 http:// 或 https:// 地址安装应用。 直接双击 HTML 文件（file://）无法安装到程序坞。 请使用下方的启动器，或按步骤手动安装。
Mac — 推荐方法
1. 下载 FamilyWealth-启动器.command
2. 双击启动（自动在 Safari 打开）
3. Safari 菜单 → 文件 → 添加到程序坞
4. 以后直接从程序坞打开即可
Mac — 手动方法
终端输入：
cd 文件所在文件夹
python3 -m http.server 7788
然后用 Safari 打开 localhost:7788
文件 → 添加到程序坞
iOS / iPad
1. Mac 启动器运行后，终端显示局域网地址
2. iPhone/iPad 与 Mac 连同一 Wi-Fi
3. 用 Safari 打开该地址
4. 分享 ⎙ → 添加到主屏幕
🌐 Chrome / Edge
启动器运行后，Chrome 打开 localhost:7788
地址栏右侧 ⊕ 图标 → 安装应用
或：菜单 → 更多工具 → 创建快捷方式
